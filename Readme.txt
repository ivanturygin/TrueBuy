// Установленные пакеты

gulp - Gulp

gulp-sass - Сборка SASS/SCSS

sass - Необходим для сборки SASS/SCSS

gulp-file-include - Подключение файлов друг в друга. Html include

gulp-clean - Удаление файлов

gulp-server-livereload - Сервер с автообновлением страницы

gulp-sourcemaps - Исходные карты для CSS

gulp-plumber - Фикс ошибок при сборке

gulp-notify - Нотификация

ImageMin - сжатие изображений

webpack -  сборщик модулей, который позволяет скомпилировать JavaScript-модули в единый JS-файл

babel - компилятор JavaScript, преобразующий код в старые версии ECMAScript

gulp-autoprefixer - добавляет префиксы к свойствам css

gulp-csso - минимизация css 

gulp-htmlclean - минимизация html

gulp-webp - конвертация изображений в формат webp

gulp-webp-html - подключение webp в html

gulp-webp-css - подключение webp в css

gulp-fonter - конвертация шрифтов

// Структура проекта


src - папка с исходниками
-----------------------------------------------------------------
html - папка с html файлами

html/components - папка с html компонентами

html/pages - папка со страницами

index.html - главный файл html (сюда подключаем все компоненты)
--------------------------------------------------------------------
img - папка с изображениями
--------------------------------------------------------------------
scss - папка со стилями

scss/base - папка с базовыми стилями

scss/base/reset - файл с обнуляющими стилями

scss/base/vars - файл с переменными

scss/base/base - файл глобальных стилей

scss/components - стили для компонентов

scss/main.scss - главный файл стилей (сюда подключаются все модули scss)
-------------------------------------------------------------------------
gulp - настройки gulp

dev.js - сборка сборка в режиме разработки

docs.js - сборка в режиме продакшена
--------------------------------------------------------------------------
js - папка с js файлами

js/modules - папка с js модулями

js/components - папка с js файлами компонентов

index.js - главный js файл 

---------------------------------------------------------------------------

build - папка с собранным проектом в режиме разработки

docs - папка с собранным проектом в режиме продакшена
